# Overview

The `mira_okf` library reads OKF bundles and the `mira-okf` CLI adapts those capabilities for human
and JSON output. Both are local and read-only. Internal implementation modules
are not part of the documented interface; commands, JSON fields, exit codes,
and supported OKF behavior are.
