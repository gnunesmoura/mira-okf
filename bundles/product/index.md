# Product

Durable product knowledge for the `tooling` library and CLI.

## Scope

This area describes the product's purpose, users, principles, and stable
capabilities. It does not contain implementation task lists or the execution
status of a current change.

## Areas

- [Features](features/) - Durable user-visible capabilities and behavior.

Product intent that applies only to one bounded implementation change belongs
in the corresponding package under
[`spec-driven-development/changes/`](../spec-driven-development/changes/).
