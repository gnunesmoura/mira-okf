# SDD Templates

Reusable templates for authoring a package under
`bundles/spec-driven-development/changes/<change-id>-<short-name>/`.

- [spec.md](spec.md) - intent, scope, behavior, and requirements.
- [plan.md](plan.md) - technical direction, boundaries, and validation.
- [tasks.md](tasks.md) - ordered, independently verifiable work units.
- [acceptance-tests.md](acceptance-tests.md) - observable acceptance checks.
- [agent-contract.md](agent-contract.md) - agent guardrails and closeout evidence.
- [log.md](log.md) - reserved chronological package history.

Replace every placeholder before advancing a package beyond `draft`.
