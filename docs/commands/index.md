# Commands

All commands are direct subcommands of `mira-okf` and use the explicit `docs` bundle
path in examples for this repository. Generic automatic discovery can be
ambiguous for this nested documentation tree. Use `--json` for machine-readable
output.

- [tree](tree.md) — summarize directories and concepts.
- [list](list.md) — inventory concepts with filters and windows.
- [show](show.md) — read one concept.
- [links](links.md) — inspect outbound links.
- [backlinks](backlinks.md) — find links pointing to a concept.
- [props](props.md) — project frontmatter properties.
- [validate](validate.md) — report OKF conformance.
- [health](health.md) — report grouped health signals.
